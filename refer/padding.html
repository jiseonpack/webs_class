padding

padding 속성은 요소의 안쪽 여백을 설정합니다.

문법

정의

  • padding-top 속성은 요소의 위쪽 바깥쪽 여백을 설정
  • padding-right 속성은 요소의 오른쪽 바깥쪽 여백을 설정
  • padding-bottom 속성은 요소의 아래쪽 바깥쪽 여백을 설정
  • padding-left 속성은 요소의 왼쪽 바깥쪽 여백을 설정
  • padding : 10px; /* 위,오른쪽,아래,왼쪽 : 10px */
  • padding : 10px 11px; /* 위,아래:10px 왼쪽,오른쪽:11px */
  • padding : 10px 11px 12px; /* 위:10px 왼쪽,오른쪽:11px 아래:12px */
  • padding : 10px 11px 12px 13px; /* 위:10px 오른쪽:11px 아래:12px 왼쪽:13px */
